# Mutants - Validation
Para desplegar el servicio se requiere docker y  docker-compose, se debe ejecutar segun el sistema operativo

####windows
```sh
.\deploy.bat
```
####Linux
```sh
sh deploy.sh
```

####Despliegue manual 
```
./gradlew clean build
cp ./app/build/libs/app.jar ./deployment/app.jar
cd deployment
docker-compose build --no-cache
docker-compose up
```
